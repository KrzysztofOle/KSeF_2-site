# KSeF_2 project website

Public, bilingual presentation website for the private **KSeF_2** project — an
evidence-driven integration bridge between KSeF 2.0 and Comarch ERP Optima.

- Website: https://krzysztofole.github.io/KSeF_2-site/
- Polish version: https://krzysztofole.github.io/KSeF_2-site/pl.html
- Hosting: GitHub Pages
- Source: static HTML, CSS and JavaScript with no external runtime dependencies

This repository contains presentation assets only. Application code, credentials,
test invoices, SQL snapshots and research evidence remain in the private project
repository.
